## 项目简介
该工程展示了如何使用[LinkSDK](https://gitee.com/LinkPi/LinkLib/wikis/pages)制作一个完整的产品，除了基本的编码串流功能以外，相比于普通的编码器设备，该工程还支持网络输入解码、转码，支持画面混合，支持文件录制、USB摄像头运动分析、文件轮播等功能。

系统提供了丰富的设置接口，用户可以根据需要裁减、定制属于自己的产品。
